## 🚀 Projetos em destaque

### 📚 Biblioteca Online - ETEFMC

Sistema web para gerenciamento de catálogo e empréstimos de uma biblioteca.

Possui funcionalidades como:
- Cadastro e remoção de livros;
- Pesquisa e filtragem por gênero;
- Sistema de empréstimos e devoluções;
- Autenticação administrativa;
- Interface responsiva com tema claro e escuro.

**Tecnologias:** HTML, CSS, JavaScript, Node.js e Express.js.

---

### 🎓 Class Manager

Sistema embarcado desenvolvido em **STM32F103C8** para controle de presença e gerenciamento de sala de aula.

O projeto utiliza:
- Display ST7789 240x240 via SPI;
- Interface gráfica embarcada;
- Controle por botões físicos;
- Registro de entradas e saídas;
- Cálculo de estatísticas da aula.

**Tecnologias:** C, STM32 HAL e sistemas embarcados.

---

### 🧩 Tesseract Paradox

Jogo de puzzles em 2D desenvolvido em JavaScript.

O jogador utiliza conceitos de dimensões paralelas para resolver desafios e alcançar o objetivo final.

**Tecnologia:** JavaScript, HTML, CSS e trabalho com lógica de puzzles.

### 🃏 SLADB

Sistema de banco de dados para armazenamento e gerenciamento de cartas de Magic: The Gathering.

Inclui:

**Tecnologias:** C#, SQL Server e .NET.
